# SmanelaKreativeCorner.com
Perpustakaan digital berbasis sekolah ini dirancang sebagai ruang kreatif bagi siswa SMA SMANELA untuk mengembangkan kemampuan menulis mereka. Melalui platform ini, para siswa dapat membagikan berbagai karya seperti cerpen, puisi, dan esai, sekaligus membaca dan mengapresiasi tulisan teman sebaya.
